⚡ Quick Start

Setup dan mulai menggunakan API dalam 5 menit.

1. Start Server

bash 
npm start

2. Create Session

bash 
curl -X POST http://localhost:3000/api/session/create \
  -H "Content-Type: application/json" \
  -d '{"sessionId": "test1"}'

3. Get QR Code

bash 
curl http://localhost:3000/api/session/test1/qr

4. Scan QR Code

📱
Cara scan:
1. Buka WhatsApp di HP
2. Menu > Linked Devices
3. Link a Device
4. Scan QR yang muncul dari response API

5. Send Message

bash 
curl -X POST http://localhost:3000/api/message/test1/send/text \
  -H "Content-Type: application/json" \
  -d '{"jid": "6281234567890@s.whatsapp.net", "text": "Hello!"}'
✅
Success! Anda sudah berhasil mengirim pesan WhatsApp via API.

Create Session

Membuat session baru untuk koneksi WhatsApp.

POST

Create Session

/api/session/create

Endpoint ini membuat session WhatsApp baru. Setelah session dibuat, Anda perlu mendapatkan QR code dan scan dengan WhatsApp di HP.

Request Body

sessionIdREQUIRED string Unique identifier untuk session (contoh: "client1", "user123")
webhookUrl string URL untuk menerima webhook events (optional)
bash 
curl -X POST http://localhost:3000/api/session/create \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "mysession1",
    "webhookUrl": "https://your-app.com/webhook"
  }'
javascript 
const response = await fetch('http://localhost:3000/api/session/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    sessionId: 'mysession1',
    webhookUrl: 'https://your-app.com/webhook'
  })
});

const data = await response.json();
console.log(data);
json 
{
  "success": true,
  "message": "Session created. Scan QR code to connect",
  "data": {
    "sessionId": "mysession1"
  }
}

Get QR Code

            
Mendapatkan QR code untuk di-scan dengan WhatsApp.


            

                

                    GET
                    
Get QR Code

                


                

                    /api/session/:sessionId/qr
                    
                


                

                    Mendapatkan QR code dalam format Data URL (base64) yang bisa langsung ditampilkan di browser.
                


                

                    
Path Parameters

                    

                        sessionIdREQUIRED
                        string
                        Session ID yang sudah dibuat
                    

                


                

                    

                        bash
                        
                    

                    
curl http://localhost:3000/api/session/mysession1/qr

                


                
Response Example:

                

                    

                        json
                        
                    

                    
{
  "success": true,
  "message": "QR code retrieved",
  "data": {
    "qr": "data:image/png;base64,iVBORw0KGg..."
  }
}

                


                

                    💡
                    

                        Tip: Gunakan QR data URL di HTML: <img src="data:image/png;base64,..." />
                    

                

            

        


        
        

            
Send Text Message

            
Mengirim pesan text ke nomor WhatsApp.


            

                

                    POST
                    
Send Text Message

                


                

                    /api/message/:sessionId/send/text
                    
                


                

                    
Path Parameters

                    

                        sessionIdREQUIRED
                        string
                        Session ID yang sudah connected
                    

                


                

                    
Request Body

                    

                        jidREQUIRED
                        string
                        WhatsApp JID (contoh: 6281234567890@s.whatsapp.net)
                    

                    

                        textREQUIRED
                        string
                        Teks pesan yang akan dikirim
                    

                


                

                    

                        bash
                        
                    

                    
curl -X POST http://localhost:3000/api/message/mysession1/send/text \
  -H "Content-Type: application/json" \
  -d '{
    "jid": "6281234567890@s.whatsapp.net",
    "text": "Hello from API!"
  }'

                


                

                    ⚠️
                    

                        Format JID: Gunakan format [country_code][number]@s.whatsapp.net

                        Contoh: 6281234567890@s.whatsapp.net (tanpa + atau -)
